About

Dit project is gefocused op het maken van een single page website waar je als je bent ingelogd verschillende mogelijkheden krijgt zoals het uploaden van stl files. Deze files worden opgeslagen in een mongoDB database. Ook kun je alle files die geupload zijn opvragen en bepaalde eigenschappen, zoals de naam, aanpassen. Deze website is gemaakt met behulp van angular op het css framework material angular.

Monster

Dit is de entiteit waarin alle informatie van een bepaald monster wordt opgeslagen:

_id Het id van het monster. (Wordt gegenereerd door Mongoose)
Name De naam van het monster
HP De hoeveelheid levens die het monster heeft
CR De challenge rating van het monster
Movement Dit is een array van alle soorten movement die het monster heeft
Creator_Id Dit is de Id van de user die dit monster heeft aangemaakt zodat alleen deze het monster kan bewerken/verwijderen
? Mogelijk komen er nog meer attributen

User

Niet iedereen mag zo maar bestanden uploaden, hiervoor moet eerst een persoon zichzelf registreren.

Username Een gebruiker heeft een username nodig
Password Een gebruiker heeft een wachtwoord nodig
Favorites? Dit is een lijst van monsters die door de gebruiker gefavoriet zijn
